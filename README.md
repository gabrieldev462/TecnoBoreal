## ◈ Sobre a TecnoBoreal

> *"Mais do que ocupar espaço, nossa vocação é gerar valor — com respeito às dinâmicas existentes, sensibilidade às necessidades específicas e dedicação à excelência em cada entrega."*

A **TecnoBoreal** dedica-se ao desenvolvimento de soluções tecnológicas adjacentes, concebidas para integrar-se de forma harmoniosa às operações de seus parceiros. Em vez de substituir estruturas centrais, buscamos fortalecê-las por meio de ferramentas que ampliam capacidades, refinam processos e promovem eficiência sustentável.

Atuamos de maneira seletiva e diligente em diferentes frentes — seja no apoio a microempreendedores por meio de aplicações voltadas à gestão e vendas, seja na criação de recursos que simplifiquem desafios operacionais.

## ◈ Padrões e Convenções

### Branches

```
main          → Produção (protegida)
develop       → Integração contínua
feat/nome     → Novas funcionalidades
fix/nome      → Correções
chore/nome    → Configurações e manutenção
```

### Commits (Conventional Commits)

```
feat:     nova funcionalidade
fix:      correção de bug
docs:     documentação
style:    formatação
refactor: refatoração de código
test:     testes
chore:    tarefas de manutenção
```

## ◈ Como Contribuir

> A TecnoBoreal opera de forma seletiva. Contribuições externas são bem-vindas mediante alinhamento prévio.

1. Abra uma **Issue** descrevendo sua proposta
2. Aguarde aprovação do time antes de abrir um PR
3. Siga os padrões de commit e branch documentados acima
4. Toda mudança passa por **code review** antes do merge
